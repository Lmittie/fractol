## Usage
```
make && make clean
./fractol [fractal]
fractals:
Mandelbrot
Julia
Newton
BurningShip
CelticMandelbrot
```
To change color press ```C```.

To move fractal use ```arrows or left mouse button(press lmb again to stop it)```.

To zoom fractal use ```mouse wheel```.

For Julia set press ```right mouse button``` to change its parameter.
